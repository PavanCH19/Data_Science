## 🐍 Phase 1: Python for Data Science (Week 1–2)

### ➤ Python Basics

* Syntax and Variables

  * Naming conventions
  * Variable types and dynamic typing
* Data Structures

  * Lists, Tuples, Sets, Dictionaries
  * Operations, indexing, slicing
* Control Structures

  * if-else, while, for loops
* Functions

  * Defining, arguments, return
  * Lambda functions
* String Manipulation

  * Slicing, methods (split, join, replace)
* List Comprehensions
* File Handling

  * Reading and writing text/CSV files

### ➤ Numpy

* Numpy Arrays

  * Creation, conversion, shape
* Indexing and Slicing
* Operations

  * Vectorized math, broadcasting
* Aggregate Functions

  * Mean, median, std, sum
* Reshape, Stack, Split

### ➤ Pandas

* Series and DataFrame
* Import/Export Data

  * read\_csv, read\_excel, to\_csv
* Indexing and Filtering

  * loc, iloc, boolean filtering
* Aggregations

  * groupby, pivot\_table
* Missing Data Handling

  * isnull, fillna, dropna
* Merging/Joining

  * concat, merge, join

## 📐 Phase 3: Statistics & Probability (Week 4)

### ➤ Descriptive Statistics

* Measures of Central Tendency (mean, median, mode)
* Measures of Dispersion (variance, std dev, range)
* Shape of Distributions (skewness, kurtosis)

### ➤ Probability

* Basic Concepts

  * Events, outcomes, sample space
* Rules of Probability

  * Addition, multiplication
* Conditional Probability
* Bayes' Theorem

### ➤ Distributions

* Types

  * Uniform, Normal, Binomial, Poisson
* PDFs vs CDFs
* Z-scores and Standardization

### ➤ Inferential Statistics

* Sampling Techniques
* Central Limit Theorem
* Confidence Intervals
* Hypothesis Testing

  * Null and Alternative Hypotheses
  * p-value, significance level
  * z-test, t-test, chi-square test

### ➤ Supervised Learning
